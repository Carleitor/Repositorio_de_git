Hay diferentes tipos de sistemas, ahora pondremos algunios

Fat (File Allocation Table)

-Medios de almacenamiento extraibles

-Sus sistemas operativos de compatibilidad son (Windows, Mac OS y Linux)

-Algunas de sus caracteristicas son su alta compatibilidad, sin funciones de cifrado ni compresion y ideal para particiones mas pequeñas.

exFAT (Extended File Allocation Table)

-Medios de almacenamiento extraible

-Con compatibilidad con Windows, Mac OS y Linux

-Con caracteristicas como tamaños y particiones ilimitasdos, no permiten gestionar derechos y no permiten comprimir los datos

NTFS (New Technology File System)

-Uso en discos duros internos y externos

- Compatible con Windows, MAC OS y Linux

- Con caracteristicas de Gestion de derechos, alto rendimiento de almacenamiento grande y permite comprimir los datos.
